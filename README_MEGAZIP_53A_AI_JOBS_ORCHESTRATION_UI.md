# MegaZIP 53A — BackOffice

Integra la vista Trabajos IA con metadatos unificados de cola y proveedor.

- Local, RunPod Serverless y Simulado en la misma pantalla.
- Cola, posición, estado del proveedor, job remoto y heartbeat.
- Métricas separadas para cola, ejecución local y RunPod.
- Cancelar y reintentar siguen usando las acciones unificadas existentes.

Los campos nuevos son opcionales para mantener compatibilidad durante el despliegue incremental.
