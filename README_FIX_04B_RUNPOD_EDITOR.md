# Fix 04B — RunPod Config Editor

Corrige la inferencia incorrecta de TypeScript en el arreglo de campos del formulario.

## Instalación

Extrae este ZIP directamente sobre:

`F:\PROYECTOS PERSONALES\TRYON\backoffice`

Acepta reemplazar el archivo.

Después ejecuta:

```powershell
Remove-Item -Recurse -Force .next -ErrorAction SilentlyContinue
npm run build
```

Si compila:

```powershell
npm run dev
```
